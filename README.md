# Frozen Stuff Firmware
Firmware for Frozen Stuff

## Hardware Requirements
 - [RP2040](https://www.raspberrypi.com/products/rp2040/)
 - [MCP3002](https://www.microchip.com/en-us/product/MCP3002)
 - [NC7WZ16P6X](https://www.onsemi.com/products/timing-logic-memory/standard-logic/logic-gates/nc7wz16) / [SN74LVC2G34](https://www.ti.com/product/SN74LVC2G34)

## Software Requirements
 - [GCC for ARM Embedded](https://developer.arm.com/tools-and-software/open-source-software/developer-tools/gnu-toolchain/gnu-rm)
 - [Pico SDK](https://github.com/raspberrypi/pico-sdk)
